Restaurant App | Documentation


------------ Customer ------------




customerFrontPage.html
pre:None
post:Customer welcomed
methods:
print welcome message
link to menu.html



menu.html
pre:Menu table exists
post:Form submitted with table number and ordered items
methods:
print heading
input table number
get Menu table from mysql
print Menu table
input quantities
submit - link to yourOrder.php



yourOrder.php
pre:Form submitted with table number and ordered items
post:Ordered items added to new Cart table by table number
methods:
print heading
create Cart table (if necessary)
insert ordered menu items into Cart table
redirect to yourOrder.html



yourOrder.php
pre:Cart table created by table number and orders added
post:Form submitted with special requests
methods:
print heading
get Cart table from mysql
print Cart table
input special requests
submit - link to yourOrderSubmitted.php



yourOrderSubmitted.php
pre:Table created by table number and orders and requests
post:Items sent to OrdersToCook table and Cart table cleared
methods:
get Cart table from mysql
get each item from Cart table
insert each item into OrdersToCook table
clear each item from Cart table
print heading
link to enjoyYourMeal.html



enjoyYourMeal.html
pre:Order sent to OrdersToCook table
post:Message displayed
methods:
display message
link to menu.html
link to yourBill.php



yourBill.php
pre:Bill table exists
post:Bill displayed and total calculated
methods:
print heading
get Bill table from mysql
calculate subtotal, tax, total
print bill
link to thankYou.php



thankYou.php
pre:Bill table exists
post:Order sent to Accounting table
methods:
print heading
get Bill table from mysql
send each item to Accounting table
delete each item from Bill table

MenuAlterationDelete.php
